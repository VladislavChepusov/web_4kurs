Курс изучения Python

Введение

Python - это высокоуровневый язык программирования общего назначения,
ориентированный на повышение производительности разработчика и читаемости кода.

Python имеет целый ряд особенностей позволяющих ему входить в топ лучших языков программирования:
  • Поддержка различных типов программирования
    • Структурное-парадигма программирования, в основе которой лежит представление программы в виде
      иерархической структуры блоков.
    • Обобщенное-парадигма программирования, заключающаяся в таком описании данных и алгоритмов,
      которое можно применять к различным типам данных, не меняя само это описание.
    • Объектно-ориентированное-парадигма программирования, основанная на представлении программы
      в виде совокупности объектов, каждый из которых является экземпляром определённого класса,
      а классы образуют иерархию наследования
    • Функциональное-парадигма программирования, в которой процесс вычисления трактуется как
      вычисление значений функций в математическом понимании последних
    • Аспектно-ориентированное-парадигма программирования, основанная на идее разделения
      функциональности для улучшения разбиения программы на модули.
  • Архитектурные черты
    • Динамическая типизация-приём используемый в языках программирования и языках спецификации, при
      котором переменная связывается с типом в момент присваивания значения, а не в момент объявления переменной.
    • Автоматическое управление памятью-периодически освобождает память, удаляя объекты, которые
      уже не будут востребованы приложениями.
    • Полная интроспекция- возможность запросить тип и структуру объекта во время выполнения программы.
    • Механизм обработки исключений-механизм языков программирования, предназначенный для описания реакции
      программы на ошибки времени выполнения и другие возможные проблемы (исключения),
      которые могут возникнуть при выполнении программы и приводят к невозможности (бессмысленности)
      дальнейшей отработки программой её базового алгоритма.
    • Поддержка многопоточных вычислений- свойство платформы (например, операционной системы,
      виртуальной машины и т. д.) или приложения, состоящее в том, что процесс, порождённый в операционной системе,
      может состоять из нескольких потоков, выполняющихся «параллельно», то есть без предписанного порядка во
      времени.
