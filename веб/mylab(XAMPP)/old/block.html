Курс изучения Python

Введение

Python - это высокоуровневый язык программирования общего назначения,
ориентированный на повышение производительности разработчика и читаемости кода.

Python имеет целый ряд особенностей позволяющих ему входить в топ лучших языков программирования:
  • Поддержка различных типов программирования:
    Структурное
    Парадигма программирования, в основе которой лежит представление программы в виде иерархической
    структуры блоков.
    Обобщенное
    Парадигма программирования, заключающаяся в таком описании данных и алгоритмов, которое можно
    применять к различным типам данных, не меняя само это описание.
    Объектно-ориентированное
    Парадигма программирования, основанная на представлении программы в виде совокупности объектов,
    каждый из которых является экземпляром определённого класса, а классы образуют иерархию наследования
    Функциональное
    Парадигма программирования, в которой процесс вычисления трактуется как вычисление значений функций
    в математическом понимании последних
    Аспектно-ориентированное
    Парадигма программирования, основанная на идее разделения функциональности для улучшения разбиения
    программы на модули.
  • Архитектурные черты:
    Динамическая типизация
    Приём используемый в языках программирования и языках спецификации, при котором переменная связывается
    с типом в момент присваивания значения, а не в момент объявления переменной.
    Автоматическое управление памятью
    Периодически освобождает память, удаляя объекты, которые уже не будут востребованы приложениями.
    Полная интроспекция
    Возможность запросить тип и структуру объекта во время выполнения программы.
    Механизм обработки исключений
    Механизм языков программирования, предназначенный для описания реакции
    программы на ошибки времени выполнения и другие возможные проблемы (исключения),
    которые могут возникнуть при выполнении программы и приводят к невозможности (бессмысленности)
    дальнейшей отработки программой её базового алгоритма.
    Поддержка многопоточных вычислений
    Свойство платформы (например, операционной системы,
    виртуальной машины и т. д.) или приложения, состоящее в том, что процесс, порождённый в операционной системе,
    может состоять из нескольких потоков, выполняющихся «параллельно», то есть без предписанного порядка во
    времени.
  • А также множество реализаций
    CPython
    Наиболее распространённая, де-факто эталонная реализация языка программирования Python
    PyPy
    Реализация Python, написанная на RPython
    IronPython
    Python для .NET Framework и Mono.
    Jython
    Реализация Python, использующая JVM в качестве среды исполнения.
  • Однако новичков в питоне может привлечь следущее...
    Python - интерпретируемый язык программирования
    С одной стороны, это позволяет значительно упростить отладку программ, с другой - обуславливает сравнительно
    низкую скорость выполнения.
    Хорошая поддержка модульности
    Вы можете легко написать свой модуль и использовать его в других программах
    Язык имеет динамическую типизацию
    Не нужно заранее объявлять тип переменной, что очень удобно при разработке.
    Встроенная поддержка Unicode в строках
    В Python необязательно писать всё на английском языке, в программах вполне может использоваться ваш родной язык.
    Автоматическая сборка мусора
    Отсутствие утечек памяти.
    Кроссплатформенность
    Программа, написанная на Python, будет функционировать совершенно одинаково вне зависимости от того, в какой
    операционной системе она запущена
